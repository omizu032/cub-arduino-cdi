# 📖 開発の軌跡と技術的進化 (Changelog / History)

* **〜 Ver 6.1**: ベースコードに対するバグ修正。多重割り込みの禁止や、遅角→進角切り替え時の不正パルス出力の修正。
* **Ver 6.2 〜 6.6 (固定小数点化の極限追求)**: 演算から割り算を追放し、`((long)sdeg * averagetime * 182) >> 16` といったシフト演算へ等価変換。
* **Ver 6.8 (量子化ノイズの撲滅と $\alpha-\beta$ フィルタの実装)**: Arduinoの4μsタイマー分解能による桁落ちを克服するため、内部変数を256倍精度へ拡張。さらに、減速時の位相遅れ（進角側への張り付き）を「2手先予測」によって数学的に相殺。
* **Ver 6.9 (5ポイントマップ＆ダブルファイア対策)**: 9000rpmまでの進角マップ拡張と、タコメーターの倍表示バグを根絶するフェイルセーフを実装。
* **Ver 7.0 ($\alpha-\beta$ 係数の定数化)**: フィルタの追従性（加減速反転時のパルスブレ抑制）を最適化するため、ゲイン係数（`ALPHA_SHIFT`, `BETA_SHIFT`）をマクロ化。実車テストによるベストバランス（$\alpha=1/2$, $\beta=1/8$）をデフォルトに設定。

※ より深いアルゴリズムの解説（2手先予測の数学的理由や、非対称予測のトレードオフなど）については、別紙 `REFERENCE.md` を参照してください。
