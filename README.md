# ホンダ スーパーカブ用 Arduino Nano デジタルCDI

ホンダ スーパーカブ90（HA02等）の横型エンジン向けに極限まで最適化された、Arduino Nano（ATmega328P）ベースのプログラマブル・デジタル進角/遅角 AC-CDI ファームウェアです。

本プロジェクトは、「[ほいほい堂blog](https://www.hoihoido.com/blog/howtomakeacdiforcub/)」の設計をベースとして、**個人的的な再構築**を行っています。

詳細は[こちら](https://zenn.dev/omizu032/articles/d4c107cd9d98df)にまとめてありますので、併せてごらんください

## 🚀 主な特徴 (Features)

* **仮想フライホイールモデル ($\alpha-\beta$ フィルタ)**
  単気筒エンジン特有の脈動（燃焼・排気ストロークの回転差）やノイズを吸収するため、整数型の定常状態推定フィルタを搭載。進角計算のジッター（ブレ）を低減しています。
* **サブディグリー（1/16度）高精度直線補間**
  進角マップの補間計算において、整数切り捨てによる「量子化（パルスの階段状の飛び）」を排除。内部的に256倍精度（固定小数点）で計算を行うことで滑らかな進角カーブを描きます。
* **5ポイント・プログラマブル進角マップ**
  1000, 3000, 5000, 7000, 9000rpm の5つの回転数ポイントで進角/遅角を設定可能。
* **究極の低レイテンシ（`ISR_NAKED` & 浮動小数点排除）**
  点火タイミング割り込みをネイティブ・アセンブラ記述（ISR_NAKED）で処理。また、32ビット除算や `float` 演算を排除して、ビットシフトと乗算のみの固定小数点演算に置き換えることで、高回転時の処理落ちやスタックオーバーフローを防いでいます。
* **レスキュー点火＆ダブルファイア防止（フェイルセーフ）**
  急激なスロットルオン時によって早くタイミングパルスが到達した際、即座にポートを直接叩いて発火させるレスキュー点火を実装。高感度なデジタルタコメーターが重複発火を拾って「回転数が倍に表示される」現象を低減するロジックも追加しています。
* **シリアルモニタによる軽量マップエディタ**
  PCやAndroidスマホ（ArduinoDroid等）のシリアルモニタ（115200bps）を接続するだけで、UIからEEPROM上のマップデータを簡単に書き換え可能です。

## ⚙️ マップの編集方法 (Usage)

1. Arduino IDE等のシリアルモニタを `115200bps` で開きます。
2. キー入力（E, C, U, D）を送信することで操作します。
   * `U` (Up) / `D` (Down): マップの選択、項目の移動、数値の増減
   * `E` (Enter): 編集モードへ入る、数値の決定（EEPROMへ保存）
   * `C` (Cancel): 編集のキャンセル
